# PalServerConfigTool

Palworld専用サーバーの設定ファイル（PalWorldSettings.ini）を作成・管理するWebアプリケーションです。

## 機能

- 全62項目のサーバー設定をカテゴリ別に編集
- PalWorldSettings.ini形式でのプレビュー表示とクリップボードコピー
- 設定のlocalStorage自動保存
- BASE64エンコードによるエクスポート/インポート（ブラウザ間データ移行）

## 使い方

`index.html`をブラウザで開くだけで使用できます。ビルドやインストールは不要です。

## 設定項目

[Palworld公式ドキュメント](https://docs.palworldgame.com/ja/settings-and-operation/configuration)に基づく設定項目に対応しています。
